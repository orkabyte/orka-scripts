# Orka Scripts

A collection of OSRS scripts for [Simba 1.4](https://github.com/Villavu/Simba).

## How to Use

1. Download and install [Simba 1.4](https://github.com/Villavu/Simba).
2. Copy the contents of the `.simba` file you want to use.
3. Paste it into Simba and hit Run.

## Scripts

| Script | Description |
|--------|-------------|
| [Orka Ammonite Crabs](orka-ammonite-crabs/) | Combat training at Ammonite Crabs with optional Birdhouse Runs |
| [Orka Campfire](orka-campfire/) | Train Firemaking 1-99 at the Grand Exchange |
| [Orka GE Cooking](orka-ge-cooking/) | Train Cooking 1-99 at the Grand Exchange |
| [Orka Lead Miner](orka-lead-miner/) | Mine lead ore on Pandemonium Island |
| [Orka Reanimation](orka-reanimation/) | Reanimate ensouled heads at the Dark Altar for Prayer XP |
